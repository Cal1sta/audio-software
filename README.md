# audio-software
针对音频的生成+采集+分析
## 生成功能
1.生成单频信号（类型、频率、相位）\
类型：正弦、矩形、脉冲等\
2.生成调制信号\
调制方式：AM、FM等\
3.生成扫频信号(给定步长、间隔时间，起止频率)\
4.要有输出功能（能把信号输出给扬声器并播放）以及保存功能
## 采集功能
输入：通过麦克风采集音频信号\
数据收集+保存\
输出：wav文件
## 分析功能
输入可以是采集到的音频，也可以是本地文件\
输出可视化(时域图，频域图)\
1.回显当前光标位置的x，y值\
2.对于频谱，x轴提供线性频率与对数频率2种选项\
3.显示音频的平均响度（RMS，均方根）
